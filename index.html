The Octogamut is a retunable webaudio keyboard

Play the Octogamut with your keyboard or mouse. Each key is labeled with its keyboard shortcut.

Change settings in the controls tab. The amplitude envelope settings for attack, decay and release are in milliseconds. Sustain is a level from 0 to 100.

Retune the Octogamut by entering values in Hz below each key. A few tunings are listed in the retune tab. The default tuning is a 6/5 Trivalent Heptatonic scale.
